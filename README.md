# PSM Generator — Affordable Tutoring Solutions

Personal Study Management tool for generating student worksheet assignments, tracking progress, and managing WellEd scores.

## 🌐 Live App

Once deployed, the app is available at: `https://<your-username>.github.io/psm-generator/`

## Features

- **Assignment Generator** — filter worksheets by subject/domain/subdomain/difficulty and generate formatted student assignments
- **Student Tracking** — enrollment, profiles, assignment history, pre-assign panel
- **Heat Map** — visual coverage of assigned domains per student
- **Score Tracking** — WellEd/BlueBook scores with trend summaries
- **Time Drill + OneNote modes**
- **Export / Import JSON** — back up your data from the header buttons

## Data Persistence

All data is saved in the browser's `localStorage` — it stays in the browser you use it in. Use the **⬇ Export** button in the header to save a JSON backup, and **⬆ Import** to restore on another device/browser.

## Running Locally

Just open `index.html` in any modern browser, or serve the folder:

```bash
python -m http.server 5173
# then open http://localhost:5173
```
